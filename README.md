1️⃣ What is the difference between var, let, and const?
var, let, and const are used to declare variables in JavaScript. var is the oldest way to declare a variable, while let and const were introduced in ES6. The main difference is that var can be re-declared and reassigned, let can be reassigned but not re-declared, and const can neither be reassigned nor re-declared.

3️⃣ What is the difference between map(), filter(), and forEach()?

The basic difference among them is that map returns a new array after transforming each element, filter returns a new array containing elements that satisfy a condition, and forEach only executes a function for each element and does not return anything.


4️⃣ What is an arrow function?

An arrow function is an ES6 feature that is used to write functions in a shorter way in JavaScript.
